这个主题已经预装并配置好了 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 集成，你可以在 astro.config.mjs 中看到相关配置。如果你暂时不打算使用 MDX，也可以把这项集成移除掉。

## 为什么使用 MDX？

MDX 是一种支持嵌入 JavaScript 和 JSX 的 Markdown 变体。它让你可以把 [JavaScript 和 UI 组件混合写进 Markdown 内容中](https://docs.astro.build/en/guides/integrations-guide/mdx/#mdx-in-astro)，从而实现交互图表、提示框之类的内容。

如果你已经有一批用 MDX 编写的内容，这个集成会让迁移到 Astro 的过程轻松很多。

## 在本项目中，你可以用 MDX 将自定义内容与目录列表组合起来，方便编写 `index.mdx`

import FolderDocList from '../../components/FolderDocList.astro';

以下为 `/blog` 的子项

<FolderDocList folder="blog" />

这是通过

```jsx
import FolderDocList from '../../components/FolderDocList.astro';

<FolderDocList folder="blog" />
```

完成的。
